🏭 आधुनिक स्टील घासणी & सह्याद्री स्टील घासणी

A fully responsive Marathi business website built for a local steel scrubber (स्टील घासणी) business.
This website helps local दुकानदार (shopkeepers) understand the services, products, and contact details easily in their native language.

📌 About the Project

This project is created for a real local business:

👤 Owner: अंकुश पवार
📍 Location: बीड, महाराष्ट्र

The business provides स्टील घासणी (Steel Scrubber) services and products to shopkeepers with a focus on:

Quality work
Timely service
Bulk supply for दुकानदार
🚀 Features
🌐 Fully Responsive (Mobile + Desktop)
🇮🇳 Marathi Language UI (Local Friendly)
🖼️ Image Gallery with Preview (Modal View)
📞 One-click Call & WhatsApp Integration
🧾 Product Information Section (Uses & Features)
⚡ Fast and Clean UI using Tailwind CSS
🔗 React Router Navigation
🛠️ Tech Stack

Frontend:

React.js
Tailwind CSS
React Router DOM
